Draft Game Instructions

    Congrats bro, you are the team's GM for a new NBA team called Las Vegas Stallions. You are able to draft free-agent players shown below (all one-year contracts). A Salary Cap will appear at the start of the game that you have to MATCH by clicking on different players to draft. Each players will have different random values during each trial of the NBA Draft Game.
    Tip: You may draft the same player(s) more than once.

  1. There are players from the table below:
    • Kristap Porzingis
    • Kyrie Irving
    • Demar De Rozan
    • Kobe Bryant
    • James Harden
    • Russell Wesbtrook
    • Lebron James
    • Steph Curry
  2. Clicking on each player will add different salary cap to your current cap.
  3. You win the game by matching your total slary to the given salary cap
  4. You have to click on the players to see the increase of value that each flag holds
  5. Each time the game is restarted, the value of players will be different as well
